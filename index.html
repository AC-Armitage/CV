Projets

Akaunting_DevSecOps

Kubernetes Sécurité DevSecOps GitHub Actions Sonarqube

Ce dépôt présente une pipeline CI/CD complète qui intègre les pratiques DevSecOps modernes afin de garantir un développement et un déploiement d'application sécurisés. L'application web Akounting, construite avec Docker, est analysée avec SonarQube pour assurer la qualité et détecter les vulnérabilités de sécurité. L'application est ensuite déployée sur Kubernetes de manière sécurisée, en utilisant GitHub Actions ainsi que les Secrets GitHub et Kubernetes pour la gestion des données sensibles. De plus, l'image Docker est scannée et approuvée par la SonarQube quality gate avant d'être déployée, garantissant que l'application respecte les normes de sécurité et de qualité tout au long du processus.
